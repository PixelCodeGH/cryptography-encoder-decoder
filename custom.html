Quick Guide

Finding Encryption Methods

  • Enter a description of what you want to do (e.g., "replace vowels with numbers")
  • Or leave it empty to get random suggestions
  • Click on any suggestion to load it into the editor

Creating Custom Methods

  • Give your method a descriptive name
  • Explain how it works in the description
  • Choose if it needs a key (e.g., a password or number)
  • Write your encryption and decryption functions
  • Test your method before saving

Function Guidelines

  • Functions must take text and key parameters
  • Return the processed text as a string
  • Avoid using unsafe functions (eval, Function, import, require)
  • Make sure encryption and decryption are reversible

Example Function

function encrypt(text, key) {
    // Your encryption logic here
    return processedText;
}

function decrypt(text, key) {
    // Your decryption logic here
    // Should reverse the encryption
    return originalText;
}
